Weather-Project 

A simple and elegant Flask-based web application that shows real-time weather information for any city using the OpenWeatherMap API.
This project is perfect for beginners learning Flask, API integration, and web development.


Features

Get live weather updates for any city
Shows temperature, humidity, weather description, and wind speed
Clean & minimal UI
API integration using OpenWeatherMap
Lightweight Flask backend application

Tech Stack

Python 3
Flask
HTML / CSS
OpenWeatherMap API
